1. Purpose and Business Context

IlluminArt is a small Polish e-commerce retailer selling art prints and decorative products. The company uses Google Ads to run Search and Shopping campaigns managed by a third-party marketing agency.

This tool is an internal, read-only reporting utility built to retrieve campaign performance data from the Google Ads API and generate bi-weekly audit reports. The reports help the business owner evaluate campaign effectiveness, identify wasted spend, and verify the quality of the agency's work.

⚠ This tool accesses only the company's own single Google Ads account. It does not manage third-party accounts and is not distributed to external users.

2. System Architecture

2.1 Components

ComponentTechnologyDescription
Data FetcherPython 3.12Scripts that call the Google Ads API and save data as JSON files
Report GeneratorAI language modelAnalyzes fetched data and produces a Markdown audit report
ConfigurationYAML fileStores account IDs and API credentials (never committed to version control)
StorageLocal filesystemRaw JSON data stored locally, never uploaded to external servers

2.2 Data Flow

  1. User triggers the audit workflow manually (approximately every 2 weeks)
  2. Python scripts authenticate via OAuth 2.0 and call the Google Ads API
  3. Retrieved data is saved locally as JSON files
  4. AI agent reads the JSON files and produces a Markdown report
  5. Report is stored locally and reviewed by the business owner

3. Google Ads API Usage

3.1 API Calls Made

ResourceOperationPurpose
campaignREAD (GoogleAdsService.Search)Campaign performance: cost, impressions, clicks, conversions, ROAS
ad_groupREAD (GoogleAdsService.Search)Ad group level performance metrics
keyword_viewREAD (GoogleAdsService.Search)Keyword performance and Quality Score
search_term_viewREAD (GoogleAdsService.Search)Actual search terms triggering ads (wasted spend analysis)
change_eventREAD (GoogleAdsService.Search)Change history — what the agency modified in the account

3.2 Operations NOT Performed

  • No campaign creation or modification
  • No bid changes or budget modifications
  • No conversion uploads or remarketing list management
  • No access to payment or billing information
  • No access to third-party advertiser accounts

3.3 Call Volume

The tool runs approximately twice per month. Each run executes 5 API queries (campaigns, ad groups, keywords, search terms, change history) for a single date range. Total monthly API calls: approximately 10 queries per month.

5. Supported Campaign Types

The tool reads data from all campaign types present in the account:

  • Search
  • Shopping
  • Performance Max

6. Summary

IlluminArt Ads Audit is a minimal, read-only internal tool designed solely to retrieve and display campaign performance data for the company's own single Google Ads account. It performs no write operations, does not manage third-party accounts, and is not distributed externally. Basic Access is requested to enable reporting on a production advertiser account.
